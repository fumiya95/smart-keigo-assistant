# ビジネス敬語 完璧マスター習得アプリ 構成定義書

本ドキュメントは、日本語教育およびUI/UXデザインの設計思想に基づき、実務に即した敬語表現を習得するための学習プロンプト構成を定義するものです。

## 1. アプリ概要
ビジネスにおける適切な言葉遣い（尊敬語・謙譲語・丁寧語）および、社内・社外の立場に応じた相対敬語を、論理的な解説と反復演習を通じて習得させる。

## 2. 学習カリキュラム

### ① 社内・上司への適切な対応
- 「ご苦労様」と「お疲れ様」の使い分け（目上への敬意）
- 上司に対する尊敬語の適用（「ご覧になる」等）
- 適切な謝罪表現（「申し訳ございません」の徹底）
- 口語表現（「あったら」「言っていた」等）の修正

### ② 三者間敬語（相対敬語）
- 外部の人間に対し、自社の上司を「身内」として謙譲語で表現する技術
- 役職名の適切な取り扱い（呼び捨て＋謙譲語の徹底）
- 紹介順序と適切な呼称の選択

### ③ 接遇言い換えマスター（NG/OK集）
- 組織の呼称（弊社/御社/貴社）
- 氏名の確認（どちら様でしょうか）
- 席外、不在時の対応
- 依頼・再確認のクッション言葉（恐れ入りますが/お手数ですが）

## 3. アプリケーション機能・ロジック

### 判定アルゴリズム
- 判定基準：正答率 90% 以上で合格
- 不合格時：誤答問題の自動再出題およびミス傾向の分類（尊敬/謙譲/立場判断）

### UI/UX 設計指針
- 色彩設計：白ベースのシンプルで高コントラストな構成
- 視覚化：矢印を用いた動作の方向性（↑ 相手を立てる / ↓ 自分を下げる）の明示
- 解説文：専門用語を避け、理由を言語化して提示

## 4. データベース定義

### 呼称管理
| 対象 | 自分側 | 相手側（社外） |
| :--- | :--- | :--- |
| 本人 | 私（わたくし） | 貴殿 / 〇〇様 |
| 会社 | 弊社 / 当社 | 御社（口頭）/ 貴社（書面） |

### 敬語変換マトリクス（基本9動詞）
- 行く / 来る / 言う / 聞く / 見る / 食べる / 会う / 知っている / もらう
- それぞれの「丁寧」「尊敬」「謙譲」を完全収録。

## 5. 運用上の重要ルール
- 二重敬語の排除。
- 慣習的な「実務敬語」を優先し、形式的な学校文法に固執しない。
- 曖昧な回答は不正解として処理し、理由の徹底解説を行う。
